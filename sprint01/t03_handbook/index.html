HTML описание

h1

Теги от h1 до h6 используются для определения заголовков HTML

<h1>Заголовок</h1>

Заголовок

p

Тег <p> определяет абзац.

Браузеры автоматически добавляют одну пустую строку до и после каждого

<p>Вот какой-то абзац</p>

Вот какой-то абзац

br

Этот тег представляет разрыв строки. Это встроенный элемент, и ему не нужен закрывающий тег.

form

Тег <form> используется для создания HTML-формы для ввода пользователем.

Элемент <form> может содержать один или несколько следующих элементов формы:

  1. <input>
  2. <label>
  3. <textarea>
  4. <button>
  5. <output>

CSS handbook

background-color

Свойство background-color устанавливает цвет фона элемента. Фон элемента — это общий размер элемента, включая отступы и границы (но не поля).

body {
background-color: rgb(1, 255, 1);
}

border

Если border-color опущен, применяемый цвет будет цветом текст.

.example {
border: 1px solid black;
}

padding

Отступ элемента — это пространство между его содержимым и его граница.

  1. padding-top
  2. padding-right
  3. padding-bottom
  4. padding-left

.example {
padding: 50px;
}

margin

Свойство margin устанавливает поля для элемента, а является сокращенным свойством для следующих свойств:

  1. margin-top
  2. margin-right
  3. margin-bottom
  4. margin-left

.example{
margin: 50px;
}

Protocols

Description

Протокол — это набор правил и указаний для передачи данных. Правила определены для каждого шага и процесса во время общения между двумя и более компьютерами. Сети должны следовать этим правилам, чтобы успешно передавать данные.

HTTP/HTTPS

Безопасный протокол передачи гипертекста (HTTPS) — это безопасная версия HTTP, который является основным протоколом, используемым для отправки данных между веб- браузер и сайт. HTTPS шифруется для увеличения безопасность передачи данных. Это особенно важно, когда пользователи передавать конфиденциальные данные, например, при входе в банковский счет, электронной почте службы или страховой компании.

DNS

DNS — это глобальная система преобразования IP-адресов в удобочитаемый доменные имена. Когда пользователь пытается получить доступ к веб-адресу, например «example.com», их веб-браузер или приложение выполняет DNS-запрос против DNS-сервера, предоставляя имя хоста. DNS-сервер принимает имя хоста и преобразует его в числовой IP-адрес, который браузер может подключиться.

IP

Адрес интернет-протокола (IP-адрес) — это числовая метка. назначается каждому устройству, подключенному к компьютерной сети, которая использует Интернет-протокол для связи. IP-адрес служит двум основным функции: идентификация хоста или сетевого интерфейса и местоположение адресация.
